⚙️ Configuración del Backend

Esta app se conecta a Google Sheets a través de un Google Apps Script. Sigue estos pasos una sola vez para activar la conexión.

1

Abre los dos Google Sheets en tu Drive

Fueron creados automáticamente. Búscalos en tu Drive:

• Inventario imac.cl — Base de Datos
• Movimientos imac.cl — Base de Datos

2

Crea un Apps Script en el Sheet de Inventario

En el Sheet de Inventario: menú Extensiones → Apps Script. Borra el código existente y pega este:

3

Despliega como Web App

En Apps Script: Implementar → Nueva implementación → Tipo: Aplicación web → Ejecutar como: Yo → Acceso: Cualquier persona → Implementar. Autoriza los permisos.

4

Pega la URL aquí

Copia la URL del despliegue (termina en /exec) y pégala abajo:
